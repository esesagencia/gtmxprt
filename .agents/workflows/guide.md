# 🤖 Everything Claude Code (ECC) - Workspace Guide

Welcome! You now have a suite of specialized AI agents and over 50 expert skills at your disposal. This guide explains how to use them effectively.

## 👥 Integrated Agents

You can invoke these agents by running their respective workflows or by simply asking me to "act as [agent name]".

| Agent | Purpose | Command |
|-------|---------|---------|
| **Planner** | Creates detailed implementation plans for complex features. | `/plan` |
| **Architect** | System design, scalability, and structural decisions. | `/architect` |
| **TDD Guide** | Expert in Test-Driven Development (Red-Green-Refactor). | `/tdd` |
| **Code Reviewer** | Reviews code for quality, security, and standards. | `/review` |
| **Security Reviewer** | Focuses on vulnerability detection and data safety. | `/security` |

## 🛠️ Key Skills

I have access to over 50 specialized skill sets. You don't need to trigger these manually; I will automatically use them when relevant to your request.

### Categories

- **Backend**: API Design, Database Patterns, Security, Performance.
- **Frontend**: React/Next.js Patterns, UI/UX Best Practices, CSS/Animations.
- **Testing**: Playwright E2E, Unit Testing, TDD Workflow.
- **DevOps**: Deployment Patterns, CI/CD, Containerization.
- **Business**: Market Research, Investor Materials, Strategic Planning.

## 🚀 Getting Started

1. **New Feature?** Start with `/plan` to get a structured breakdown.
2. **Design Question?** Ask `/architect` for structural advice.
3. **Finished Coding?** Use `/review` to ensure everything meets production standards.
4. **Need Tests?** Use `/tdd` to implement features with high coverage.

Would you like to try one of these right now? Just type the command!
