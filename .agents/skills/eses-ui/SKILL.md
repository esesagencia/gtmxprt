# ESES UI System Brand Standards

Use this skill to ensure all code and UI elements generated by Antigravity adhere to the ESES Agency visual system.

## 🎨 Color Palette

- **Brand Boreal**: `#5DFF78` (Primary/Action)
- **Brand Carbon**: `#1A1A1A` (Text/Emphasis)
- **Brand Polar**: `#A855F7` (Secondary/Immersive)
- **Brand Slate**: `#F8FAFC` (Background/Neutral)

## ✍️ Tipografía y Jerarquía HTML

- **Headings (h1-h6)**: Siempre clase `font-display` ("Founders Grotesk"), `font-bold`, y `tracking-tight`.
  - `h1`: Texto muy grande, ej. `text-5xl` o `text-6xl`.
  - Preferible usar *lowercase* con capitalización en la primera letra (`first-letter:uppercase`) para dar el toque distintivo de la agencia.
- **Body Text (p, span)**: Siempre clase `font-sans` ("Open Sans").
  - Peso: `font-light` o `font-medium`.
  - Interlineado amplio para legibilidad: `leading-relaxed` (1.625).
  - Texto secundario: Usa opacidades sobre el color principal, ej. `text-brand-carbon/60` en lugar de grises genéricos.

## 📐 Estructura HTML, Márgenes y Layout (CSS/Tailwind)

- **Grid y Flexbox**: Usa siempre `flex` o `grid` para el layout. Gap mínimo recomendado: `gap-6` (24px) o `gap-8` (32px).
- **Paddings Internos (Respiración)**: Las tarjetas y secciones deben tener mucho espacio en blanco. Usa `p-8` o `p-10`.
- **Radios de Borde (Border Radius)**:
  - Para botones, inputs y tags: `rounded-xl` o `rounded-2xl`.
  - Para tarjetas contenedoras principales: `rounded-[2rem]` (32px) u proporciones orgánicas mayores.
- **Bordes Funcionales**: Usa bordes hiper-finos para delimitar sin ensuciar visualmente: `border border-brand-carbon/5` o `border-white/10`.
- **Sombras (Shadows)**: Prohibido usar las sombras por defecto de Tailwind en negro puro. Usa siempre sombras difuminadas con el color de acento, ej: `shadow-xl shadow-brand-boreal/10`.

## 🛠️ Component Usage

Always prefer the custom components in `src/eses-ui/components/`:

- `Button`: Physics-based magnetic buttons.
- `cn`: Utility for class merging.

## 🚀 Principles

1. **Motion First**: Every interaction should have a `framer-motion` transition.
2. **Global Smoothness**: Lenis is mandatory for page scroll.
3. **Immersive Feedback**: Subtle glows and glassmorphism (backdrop-blur).
